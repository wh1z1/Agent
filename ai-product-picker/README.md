# 🤖 AI 选品官 — AI-Powered E-commerce Product Picker

> 从趋势发现到一键上架的全自动电商 Agent 系统

```

                        │ REST API
┌───────────────────────┴─────────────────────────┐
│               FastAPI 后端 (Backend)              │
│                                                  │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐        │

│       │            │            │               │
│  ┌────┴─────┐ ┌────┴─────┐ ┌────┴─────┐        │

```

## 快速启动

```bash
cd backend
pip install -r requirements.txt
python main.py
# 访问 http://localhost:8000
```

## 5 大 Agent

| Agent | 职责 | 输出 |
|-------|------|------|
| 🔍 趋势猎手 | 扫描全网热搜，发现潜力品类 | 每日 Top 5 选品推荐 |
| 🏭 供应链 | 1688找厂、比价、询价 | 供应商报价单 |
| ✍️ 内容工厂 | 标题/文案/主图/短视频脚本 | 完整上架素材包 |
| 🚀 上架运营 | 多平台一键发布 | 店铺已上架 |
| 📊 数据驾驶舱 | 实时监控+自动优化 | 日报 + 自动干预 |

## 技术栈

- **后端**: Python 3.11 + FastAPI
- **前端**: 原生 HTML/CSS/JS（零依赖）
- **Agent**: 自研轻量编排框架
- **数据**: 内存模拟（可扩展至 Redis/PostgreSQL）
